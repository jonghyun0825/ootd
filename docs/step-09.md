# Step 09. 필터/수정/삭제/백업 기능 완성

## 목표
- 필터 화면에서 실제 DB 데이터를 대상으로 계절/카테고리/아이템종류/색상/자유태그 AND 조건 필터링 구현
- 상세 화면에서 실제 데이터 표시, 수정 화면에서 DB 업데이트 반영
- 삭제 시 DB(사진+아이템) 및 Storage(큰 이미지+썸네일) 동시 삭제, 삭제 전 확인 모달
- JSON 백업 내보내기/가져오기 구현 (`fashion-archive-backup-YYYY-MM-DD.json`)
- 전체 데이터 삭제 기능(강한 확인 절차 포함)

## 완료 기준
- [ ] 필터 조합 적용 시 조건에 맞는 사진만 표시, 결과 없음 문구 정상 동작
- [ ] 수정 후 상세 화면에 변경 사항 즉시 반영
- [ ] 삭제 시 Storage 파일도 함께 삭제됨을 Supabase 콘솔에서 확인
- [ ] JSON 백업 파일 다운로드 및 재가져오기 성공
- [ ] 전체 삭제 확인 문구 노출 및 동작 확인
