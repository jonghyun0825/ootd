# Step 07. Supabase Auth 실제 연동

## 목표
- 이메일 매직 링크(Magic Link) 로그인을 실제 Supabase Auth와 연동한다.
- 로그인 상태에 따라 로그인 화면 / 앱 본문을 분기한다.
- 로그아웃 기능을 아카이브 화면에 연결한다.
- 세션 유지(새로고침 후에도 로그인 상태 유지) 처리.

## 사용자 작업 필요
- Supabase 프로젝트의 URL / anon key를 `.env`에 입력 (정확한 위치는 supabase-setup.md 참고)
- Supabase Authentication 설정에서 Email 로그인(매직링크) 활성화 확인

## 완료 기준
- [ ] 실제 이메일로 매직 링크 수신 및 로그인 성공
- [ ] 새로고침해도 로그인 상태 유지
- [ ] 로그아웃 시 로그인 화면으로 복귀
- [ ] 로그인하지 않은 사용자는 앱 데이터 화면에 접근 불가
